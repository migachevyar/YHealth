# YHealth — Telegram Mini App

Промпт для воссоздания: создай Telegram Mini App для трекинга здоровья YHealth.

## Стек
- Python 3.11, python-telegram-bot==20.7, Railway хостинг
- WebApp: HTML/CSS/JS в одном файле, без фреймворков
- Данные: Telegram CloudStorage (основное) + SQLite на сервере (для бота)
- Иконки: Phosphor Icons v2.1.1, Шрифт: Inter

## Переменные Railway
BOT_TOKEN, WEBAPP_URL, FEEDBACK_CHAT_ID, TZ_OFFSET=3

## Дизайн
Тёмная тема (#09090f), стиль Apple Fitness. Акцент: зелёный #2fd158, синий #0a84ff, оранжевый #ff9f0a, фиолетовый #bf5af2. Полный экран через tg.requestFullscreen().

## Онбординг (4 шага)
1. Имя + Цель (Сбросить вес / Набрать мышцы / Поддержать форму)
2. Пол / Возраст / Вес / Рост
3. Время завтрака → автоматический расчёт расписания (ужин макс 19:30). Редактируемый preview.
4. Выбор витаминов (10 шт, рекомендованные по профилю)

## 4 вкладки
**Сегодня:** 3 кольца прогресса (питание/вода/витамины), карточка воды (5 бутылок 0.5л), список приёмов пищи с галочками, витамины с галочками, лекарства

**Питание:** КБЖУ по Миффлину (коррекция по цели), примеры белка, варианты блюд по дням (ротация), справочник витаминов

**Статистика:** серия дней, сетка 2×2, графики за 7 дней, история веса с SVG-графиком и удалением

**Профиль:** расписание (time input + тогл + каскадный пересчёт при изменении завтрака), витамины (тогл + время + добавить), лекарства (цветные иконки + time input + удаление), замечания, перенастройка

## Хранение
CloudStorage: `profile`, `day_YYYY-MM-DD`, `weights` (массив [{date,value,ts}])
SQLite: таблица user_data(uid,key,value), ключ `schedule` для бота

## Бот
/start, /remind (читает schedule из SQLite, группирует события по времени, случайные тексты), /stop, /help
Конвертация времени: utc = local - TZ_OFFSET*60

## Витамины (10 шт, время расчётное)
omega(завтрак), vitd(завтрак), magnesium(22:00), vitc(завтрак через день), zinc(обед), creatine(любое), vitb12(завтрак), iron(за 30 мин до завтрака), calcium(обед), probiotics(за 30 мин до завтрака)
Несовместимости: iron не с zinc/calcium, vitc не с vitb12 при высоких дозах

## Синхрон расписания
pushScheduleToServer() вызывается при: финише онбординга, смене времени приёма, тогле приёма, добавлении/удалении лекарства
